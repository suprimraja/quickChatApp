# Code of Conduct

## Our Pledge

We commit to making participation in this project a harassment-free experience for everyone.

## Our Standards

- Be respectful and considerate.
- Use welcoming and inclusive language.
- Avoid personal attacks or discriminatory remarks.
- Value diverse perspectives.

## Enforcement

Instances of abusive behavior may be reported by opening an issue or contacting the repository maintainers.

This project follows the Contributor Covenant Code of Conduct.
